# 🚀 CalcVerse

Welcome to CalcVerse – an all-in-one mathematical toolkit built using Python and Streamlit.

CalcVerse combines multiple powerful mathematical tools into a single interactive platform, making calculations, graph visualization, statistical analysis, and equation solving simple and efficient.

🌐 Live Demo: https://huggingface.co/spaces/Bandana17/ai-calculator

---

## ✨ Features

🧮 Scientific Calculator

📈 Interactive Graph Plotter

📊 Statistical Analysis Tools

🧠 Equation Solver

📷 OCR-Based Calculator

📜 Calculation History Tracking

🎨 Clean and Modern User Interface

⚡ Fast Real-Time Computation

## 🚀 How It Works

1. Select a tool from the sidebar.
2. Enter your mathematical expression or data.
3. Process the input instantly.
4. View results, graphs, statistics, or solutions.
5. Save and review calculation history.
